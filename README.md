# NewsApp

The app present news from https://newsapi.org.

 Architecture: MVP
 
 libraries: Foundation, UIKit, CoreData
 
 Pods: Alamofire, ObjectMapper, PromiseKit, Kingfisher

## Features

* You can add news channels to favorites list
